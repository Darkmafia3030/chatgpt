# Dark LAN App

### How to Use
1. Run `start-app.bat`
2. Access `http://localhost:3000`
3. Admin: `admin` / `Ukm@#04`
4. User: `user` / `user1`

### Features
- Login system with admin/user
- Excel-style dashboard (4 tabs)
- IP-based access control
- SQLite + CSV/XLSX support
- Print selection
- Import/Export
